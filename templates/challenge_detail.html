Common Mistakes & Troubleshooting

Problem: Tool/command is not installed
Solution: Install the missing tool using apt (Linux) or brew (Mac) 
sudo apt install tool-name

Problem: You don't have permission to run the command
Solution: Use sudo to run with elevated privileges 
sudo command-name

Problem: Cannot connect to target
Solution: Check if VM is running and correct IP is used 
ping target-ip
nmap -sn target-ip

Network Security
Command Purpose

Hydra - Password Cracking

Cryptography
Dictionary Attack (MD5)

Brute Force
hashcat -m 0 -a 3 hashes.txt ?a?a?a?a

John the Ripper
Cryptography
Dictionary Attack
